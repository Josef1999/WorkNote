- 删除光标所处至指令串尾 [ctrl] + [k] 

  

## shell

#### bash shell功能

- **指令记录** ~/.bash_history 上一次登录以前执行过的指令（每次退出登录后更新）

- **指令补齐** [tab] 补齐指令或文件名

- **alias别名** alias ll = 'ls -l' 自定义指令

  

#### 相关指令

- **type [-tpa] name** 查询name的指令类型

- **env** 列出所有环境变量

- **set** 列出所有环节变量、bash变量

- **locale** 显示所用编码
