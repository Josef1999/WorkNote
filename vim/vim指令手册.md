# vim指令手册

## 移动光标

### 按位移动

- 向上一个 [↑]

- 向下一个 [↓]

- 向左一个 [←]

- 向右一个 [→]

  

### 按页移动

- 向上一页 [Page Up]

- 向下一页 [Page Down]

- 向上半页 [Ctrl] + [u]

- 向下半页 [Ctrl] + [d]

  

### 按行移动

- 向上一行 [=]

- 向下一行 [-]

- 向下n行 n + [Enter]

- 移动到第n行 n + [G]

- 移动到第一行 [g] + [g]

- 移动到最后一行 [G]

  

### 特殊移动

- 移动到当前行首 [0] / [Home]
- 移动到当前行尾 [$] / [End]
- 移动到当前屏幕首行行首 [H]
- 移动到当前屏幕中间行行首 [M]
- 移动到当前屏幕末行行首 [L]



## 搜索替换

### 搜索

- 从光标之后搜索 /word
- 从光标之前搜索 ?word
- 搜寻下一个字符串 n
- 搜寻上一个字符串 N



### 替换

- :n1,n2s/word1/word2/g  在n1-n2行之间用word2替代word1

- :1,$s/word1/word2/g  在首行到末行间用word2替代word1

- :1,$s/word1/word2/gc 在首行到末行间用word2替代word1，替代前由用户确认

  

  

## 删除

### 按字符删除

- 删除前一个字符 [x] / [Delete]
- 删除后一个字符 [X]
- 删除前n个字符 n + [x]
- 删除后n个字符 n + [X]
- 删除光标所在处至行尾 [d] + [$]
- 删除光标所在处至行首 [d] + [0]



### 按行删除

- 删除光标所在行 [d] + [d]
- 删除光标所在及之下n行 n + [d] + [d]
- 删除光标所在行至首行 [d] + [1] + [G]
- 删除光标所在行至末行 [d] + [G]



## 复制黏贴

### 按字符复制

- 复制光标所在处至行尾 [y] + [$]
- 复制光标所在处至行首 [y] + [0]

### 按行复制

- 复制光标所在行 [y] + [y]
- 复制光标所在及之下n行 n + [y] + [y]
- 复制光标所在行至首行 [y] + [1] + [G]
- 复制光标所在行至末行 [y] + [G]

### 黏贴

- 黏贴在光标所处下一行 [p]
- 黏贴在光标所处上一行 [P]



## 特殊操作

- 合并光标所处行及下一行 [J]

- 撤销上一次操作 [u]

- 重复上一次操作 [Ctrl] + [r] / [.]

  

## 多文件编辑

**需打开多文件 例：vim file1 file2**

- 编辑下一个文件 :n

- 编辑上一个文件 :N

- 显示所有当前vim打开的文件 :files

- 编辑第n个文件 :bn

  

## 多视图编辑

### 创建视图

- 创建本文件视图 :sp

- 创建其他文件视图 :sp filename

  

### 切换视图

- 上一个视图 [Ctrl] + [w] + [↑]

- 下一个视图 [Ctrl] + [w] + [↓]

  

### 关闭视图

- 关闭当前视图 [Ctrl] + [w] + [q]
